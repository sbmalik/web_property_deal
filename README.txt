How To Run:
- install python 3.8
- open terminal into extracted folder
- type `pip install -r requirements.txt`
- type `python manage.py runserver`
- open browser and type `http://127.0.0.1:8000/`
- for admin panel visit `http://127.0.0.1:8000/admin`
- you can add new houses or users from admin panel (can modify/update their values)
